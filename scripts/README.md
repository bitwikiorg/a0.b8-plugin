# Scripts Documentation

This directory contains utility scripts for setting up and managing the Bithub environment.

## setup_env.py
Initializes the local development environment. It handles:
- Creation of necessary directories (e.g., for logs or data).
- Validation of environment variables.
- Initial configuration generation.

## setup_mcp.py
Configures the Model Context Protocol (MCP) integration. It:
- Sets up the MCP server connections.
- Validates MCP configuration files.
- Ensures the agent can communicate with external MCP tools.
