# AGENTS.md

## Context
This is the Bithub Client System.

## Dev Tips
- Ensure you are using Python 3.8+.
- Activate your virtual environment (venv) before development.
- Install the package in editable mode: pip install -e .
- Use scripts/setup_env.py for environment initialization.

## Testing
- Run the test suite using pytest.

## PRs
- Always check and update the version in pyproject.toml before submitting a Pull Request.

## Core Concepts
- **Core**: A specialized AI workflow triggered by the **First Post** in a designated subcategory (e.g., Hegel, Aletheia). It represents a heavy analytical process.
- **Core Seed**: The **Output** generated by the Core workflow (the First Reply). This immutable seed serves as the foundation for subsequent multi-agent discussion.
